Wild Life

An Android application on Google Play for your personal growing wildlife encyclopedia. It encourages you to add photography to the database, and to grow and learn more about the species around you. Obseve and learn more about your personal wild life.

Foods R Us

A fully functional eCommerce website built with Java, Javascript, and SQL, with user authentication.

Users can browse through a dynamic catalog of items that are stored and obtained from an SQL database. Items can be placed on a cart for checkout. Orders are stored in XML sheets. Users can log in through an account before checking out.

This project taught me the amount of detail needed to ensure that a customer's security is safe. Our team decided to use HTTP authentication, where the user would use his York University log in credentials. It also taught me the importance of scalability in an eCommerce website. Any kind of item can be displayed, as such its descriptions need to be formatted and displayed correctly. The items could change at any time.

Distance Sensor

A Distance Sensor designed a C, using a microcontroller to detect the distance of a object, and emit a sound frequency the closer the object gets.

Using an IR Rangefinder, any object's distance can be calculated. The distance the object is at is displayed on the Seven-segment display. A beep is emitted in short intervals as well. As the object moves close to the sensor, the beep frequency increases. You can think of it as as a backup sensor in a vehicle. When the object is extremely close, a strong beep from the LED is emitted, warning the user that the object is really close.

This project taught me a great deal about working with real time systems. Since the system is continuously running, task management is essential. Just like multi-threaded applications, we had to ensure global variables are properly protected with mutexes, such that two processes are not accessing the same resource at the same time.
